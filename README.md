# 🍎 Contador de Calorias - Calorie Counter

Um **aplicativo web moderno** para rastrear sua ingestão de calorias diária com suporte a **11 idiomas** e **tema claro/escuro**.

## 🌐 Acesse Agora!

**[🚀 Clique aqui para usar o app!](https://d124r.github.io/meu-primeiro-programa-em-Js-JavaScript/)**

---

## ✨ Recursos

✅ **Contador de Calorias em Tempo Real** - Adicione alimentos e veja o total instantaneamente  
✅ **Estimativa Inteligente** - Insira a descrição do alimento (frito, cozido, com azeite) e a app estima as calorias  
✅ **11 Idiomas** - Português, English, Español, Français, Deutsch, Italiano, 日本語, 中文, Русский, العربية, 한국어  
✅ **Tema Claro/Escuro** - Toggle entre tema claro (☀️) e escuro (🌙)  
✅ **Banco de Dados Local** - Usa IndexedDB para armazenar dados offline  
✅ **Persistência** - Seus dados e preferências são salvos automaticamente  
✅ **Design Responsivo** - Funciona em desktop, tablet e mobile  
✅ **Zero Dependências** - Apenas HTML5, CSS3 e JavaScript puro

---

## 🚀 Como Usar

### 1. Acesse o App
Abra em qualquer navegador: https://d124r.github.io/meu-primeiro-programa-em-Js-JavaScript/

### 2. Adicione Alimentos
- Digite o nome do alimento (ex: Maçã)
- (Opcional) Descreva como foi preparado (ex: frita, com azeite)
- Clique em "Estimar calorias" ou insira manualmente
- Clique em "Adicionar entrada"

### 3. Acompanhe seu Progresso
- Veja o total geral de calorias
- Selecione uma data para ver calorias do dia específico
- Remova entradas quando necessário

---

## 🎨 Personalização

**Trocar Idioma:** Clique no dropdown "Idioma" no canto superior direito  
**Trocar Tema:** Clique no botão 🌙/☀️ para alternar entre claro e escuro

Ambas as preferências são salvas automaticamente!
